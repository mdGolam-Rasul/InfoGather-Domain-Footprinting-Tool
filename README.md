## ⚙️ Features

- ✅ WHOIS lookup — domain owner, registrar, dates
- ✅ DNS records — A, NS, MX, TXT
- ✅ Geolocation — country, city, lat/long
- ✅ Shodan IP lookup — open ports, banners, org
- ✅ Save results to a file
